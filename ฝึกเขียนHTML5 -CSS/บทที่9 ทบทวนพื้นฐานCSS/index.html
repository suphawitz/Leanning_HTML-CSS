คณะเจ้าหน้าที่โรงพยาบาลเสริมงาม

คณะเจ้าหน้าที่โรงพยาบาลเสริมงาม ขอขอบพระคุณ คุณกิตติพงษ์(เจ๋ง) ยะเปียง เจ้าของนะโมฟาร์ม (เมล่อนฟาร์ม) บ้านสบเสริม ที่ได้มอบเมล่อนเพื่อเป็นขวัญและกำลังใจแด่เจ้าหน้าที่ทุกคนที่ต้องต่อสู้และให้บริการผู้ป่วยในช่วงที่ไวรัสโคโรน่าระบาดในช่วงนี้ คณะเจ้าหน้าที่ทุกๆคนต้องขอขอบคุณมา ณ โอกาสนี้ครับ
