width: ; จะคงค่าไว้แบบเดิม ไม่มีการขยับหรือเปลี่ยนแปลงขนาดไปจากที่กำหนด

position

จัดตำแหน่งของกล่องข้อความ

position: static; static เป็นค่าปกติอยู่เเล้ว
position: relative; จัดตำแหน่งจากการอ้างอิงตำแหน่งเดิม
position: fixed; เมื่อสกอร์เมาส์จะไปเคลื่อนตาม
position: sticky; เมื่อสกอร์เมาส์ผ่านจะเคลื่อนที่ตามลงจากตำแหน่งเดิม
position: adsolute; adsolute ทำงานร่วมกับ relative ในการกำหนดกล่องขึ้นมาโดย adsolute จะเป็นกล่องย่อยที่อยู่ในกล่องของ relative

hidden เนื้อหาที่เกินออกมาจะถูกตัดออกไป
acroll ข้อความทั้งหมดจะอยู่ในกล่อง โดยมีการสกอร์บาร์ขึ้นมา
